# predictive-k8s-autoscaling

## Repository layout
- `core/`: shared datasets, models, and utilities.
- `pipelines/`: runnable end-to-end workflows (full, local, preprocessing-only).
- `preprocessing/`: data fetching, ingestion, and window generation.
- `training/`: training and evaluation entry points.
- `tooling/`: supporting tools (weights computation, tuning).
- `analytics/`: experiment analysis and calibration scripts.
